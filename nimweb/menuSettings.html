Menu Settings

The currently exist two settings: the File and the Strategy. Use File when you want to perform actions on the highest level of the application. Use Strategy when you want to change the strategy the Computer you are playing against uses.

File

  • Exit (Ctrl + x): Exit will close the application. You can also Exit by using the shortcut Ctrl + X

Strategy

  • Cautious: The computer plays conservative. This is the default when you play
  • Greedy: The computer plays greedy or more aggressive
  • Random: The computer plays unpredictably. Good setting to use when you want a more simulated style of play
