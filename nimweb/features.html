Features

To allow for a more immersive gameplay, some new features have been added. It is the goal to continuously add new features. Below are the ones that currently exist.

Features and Specs:

  • Flexible Pile: Upon load of the game (before selecting a player), you can adjust the size of the pile for the game
  • Coin Flip: Can't decide who should go first? Coin Flip will begin the game and randomly select which player will go first
  • Automated Computer Player: No need to manually choose the computer to go. The computer is now automated to known when to take its turn
  • Quick Think Timer: You don't have all day! You now have a time limit to take your turn so think quick!

New updates are added regularly so check back often!
